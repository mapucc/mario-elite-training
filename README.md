# MARIO ELITE TRAINING PWA

**6-Day Elite Hybrid — Arm Thickness Phase**

A Progressive Web App for iOS built for Mario's 12-week training program.

## Features
- Auto-detects today's training day on launch
- Set tracker — tap dots to log each set, cards turn green when complete
- Live rest timer that auto-fires when you log a set, turns red at 10 seconds
- Bonus exercise blocks (Glute Finisher Mondays, DB Pullover Wednesdays)
- Rest Day recovery guidance with sleep protocol
- Reference mode + step-by-step Guided mode
- Full nutrition framework, guardrails, supplement stack
- Periodization, double progression rules, pull-up protocol
- Full offline support via Service Worker
- Black/gold/rust aesthetic

## Install on iPhone
1. Open the GitHub Pages URL in **Safari**
2. Tap the **Share** button (box with arrow)
3. Tap **"Add to Home Screen"**
4. Name it **MARIO ELITE** → tap **Add**

## GitHub Pages Deployment

1. Create a new repo on GitHub (e.g. `mario-elite-training`)
2. Upload all files: `index.html`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`
3. Go to **Settings → Pages → Source → main branch → / (root)**
4. Your PWA is live at `https://yourusername.github.io/mario-elite-training/`
